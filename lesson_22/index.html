Задача №3

Що потрапить в консоль?

if (2 * 20 <=10 || 30 / 2 < 5 && 10 <= "10" || 20==="20" ) {

console.log('000') }

Відповідь:

В консоль нічого не потрапить, тому що умова поверне значення false.

Виконаємо обчислення по- порядку:

  • 1.Оператори порівняння мають вищій приоритет
    1. 1.1 2 * 20 <= 10 - false
    2. 1.2 30 / 2 < 5 - false
    3. 1.3 10 <= "10" - true
    4. 1.4 20 === "20" . Оператор суворого порівняння "===" не змінює тип данний, тому - false
  • Отримуємо вираз : if (false || false && true || false)
  • 2.Далі логічні оператори. Приоритет має оператор "&&"
  • Отримуємо вираз : if (false || false || false)
